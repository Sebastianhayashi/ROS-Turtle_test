# 仓库描述

该仓库是用于放置在 openEuler 24.03 上的小乌龟测试。

## 测试描述

系统版本：openEuler 24.03

参考[该文档](https://gitee.com/openeuler/ros/tree/master/user_doc/ROS-humble-oerv24.03-x86)在 openEuler 24.03 上安装 ROS2 Humble 并且测试能否正常移动小乌龟。
